# booking-app

## Prerequisite

- Node.js
- Redis
- MongoDB

## Installation

#### 1. Clone project (BE)

```gitbash
git clone https://github.com/minhtri06/booking-app.git
cd booking-app
```

#### 2. Install packages

```gitbash
npm install
```

#### 3. Create file .env based on .env.example

Input port, database connection

#### 4. Generate data

```gitbash
npm run seed
```

#### 5. Start server

```gitbash
npm start
```
